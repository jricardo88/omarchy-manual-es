# AI

Omarchy viene con [OpenCode](https://opencode.ai/) y [Claude Code](https://code.claude.com/docs/en/overview) como los arneses de agentes predeterminados para programadores. OpenCode te permite usar modelos de todos los principales proveedores comerciales, así como modelos open-weight que mejoran rápidamente. La mejor forma de usar OpenCode es yendo al directorio en el que vas a trabajar en la terminal, luego invocando el alias `c` para iniciar OpenCode dentro de ese directorio. Claude Code te permite usar los modelos de Anthropic con sus suscripciones propietarias, y puede iniciarse en modo danger usando el alias `cx`.

## Dictado

Omarchy ofrece dictate por AI via [Voxtype](https://voxtype.io/). Lo instalas via *Instalar > AI > Dictado* a través del menú de Omarchy. Por defecto, cargará un modelo base en inglés que ocupa 150MB. Pero puedes ajustar qué modelo quieres usar ejecutando `voxtype setup model` en la terminal. Y puedes ajustar todas las configuraciones via `~/.config/voxtype/config.toml`.

Una vez instalado, dictas manteniendo presionado `Super + Ctrl + X`, y una vez que lo sueltas, el texto dictado aparecerá en el área de entrada con foco.

## La Skill de Omarchy

Las skills de agentes ayudan a la AI a usar herramientas específicas de forma específica, y Omarchy viene con una skill por defecto para personalizar el sistema. Como hacer cambios a waybar o incluso crear un nuevo tema desde cero. Está configurada automáticamente en OpenCode (y Claude Code y cualquier otro arnés que soporte `~/.claude/skills`).

Pero deberías tratar esta skill como experimental. Diferentes modelos la usarán con diferentes efectos. Es mejor ejecutar primero en modo plan, así tienes una idea de lo que el agente gostaria de cambiar. Y luego estar listo para revertir cambios o incluso invocar `omarchy-reinstall-configs`, si el agente hace un desastre de todo.

## LLMs locales

Omarchy recomienda dos formas de ejecutar modelos LLM locales: LM Studio y Ollama. LM Studio proporciona una interfaz GUI para encontrar modelos open-weight, instalarlos y ejecutarlos. Es una gran forma de empezar fácilmente. Ollama ofrece una CLI para hacer lo mismo. Pero si eres nuevo en modelos locales, empezaría con LM Studio. Puedes instalar cualquiera bajo *Instalar > AI* en el Menú de Omarchy.

## Otros arneses de agentes

Si no quieres usar OpenCode como un arnés multi-modelo o Claude Code por cualquier razón, también puedes instalar los arneses específicos de proveedores desde *Instalar > AI*. Encontrarás Gemini CLI, OpenAI Codex, GitHub Copilot aquí.

También tienes acceso a Crush, que es una alternativa a OpenCode por el excelente equipo de Charm (que hace la herramienta TUI `gum` con la que está construido mucho de Omarchy).
