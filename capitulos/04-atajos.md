# Atajos de teclado

Puedes ver todos los atajos de teclado principales con `Super + K`.

## Navegación

| Atajo | Función |
|-------|---------|
| `Super + Space` | Lanzador de aplicaciones |
| `Super + Alt + Space` | Menú de control de Omarchy |
| `Super + Escape` | Menú del sistema (suspender, reiniciar, etc.) |
| `Super + Ctrl + L` | Bloquear computadora |
| `Super + W` | Cerrar ventana |
| `Ctrl + Alt + Del` | Cerrar todas las ventanas |
| `Super + T` | Alternar ventana entre tiling/floating |
| `Super + O` | Alternar ventana a sticky'n'floating |
| `Super + L` | Alternar entre layout dwindle y scrolling |
| `Super + F` | Pantalla completa |
| `Super + Alt + F` | Ancho completo |
| `Super + 1/2/3/4` | Ir a workspace específico |
| `Super + Tab` | Ir al siguiente workspace |
| `Super + Shift + Tab` | Ir al workspace anterior |
| `Super + Ctrl + Tab` | Ir al workspace anterior (anterior actual) |
| `Super + Shift + 1/2/3/4` | Mover ventana al workspace |
| `Super + Shift + Alt + Flechas` | Mover workspaces a monitor direccional |
| `Super + Flecha` | Mover foco a ventana en dirección de flecha |
| `Super + Shift + Flecha` | Intercambiar ventana con otra en dirección de flecha |
| `Super + Equal` | Crecer ventanas a la izquierda |
| `Super + Minus` | Crecer ventanas a la derecha |
| `Super + Shift + Equal` | Crecer ventanas hacia abajo |
| `Super + Shift + Minus` | Crecer ventanas hacia arriba |
| `Super + G` | Alternar agrupación de ventanas |
| `Super + Alt + G` | Mover ventana fuera de agrupación |
| `Super + Alt + Tab` | Ciclar entre ventanas en agrupación |
| `Super + Alt + 1/2/3/4` | Ir a ventana específica en agrupación |
| `Super + Alt + Flecha` | Mover ventana a agrupación en dirección de flecha |
| `Super + Ctrl + Flecha` | Mover entre ventanas dentro de grupo tiling |
| `Super + S` | Mostrar overlay del workspace scratchpad |
| `Super + Alt + S` | Mover ventana al workspace scratchpad |
| `Super + Ctrl + Z` | Zoom in en pantalla (repetir para más zoom) |
| `Super + Ctrl + Alt + Z` | Zoom out completo de pantalla |
| `Super + /` | Ciclar entre opciones de escala de monitor |

## Portapapeles universal

| Atajo | Función |
|-------|---------|
| `Super + C` | Copiar |
| `Super + X` | Cortar (no en terminal) |
| `Super + V` | Pegar |
| `Super + Ctrl + V` | Gestor de portapapeles |

Por lo general en Linux necesitas `Ctrl + Shift + C/V` para copiar y pegar en la terminal y `Ctrl + C/V` para hacerlo en todos lados. Estos atajos unificados de portapapeles de Omarchy funcionan en todas partes (excepto en el administrador de archivos).

## Captura

| Atajo | Función |
|-------|---------|
| `Super + Ctrl + C` | Menú de captura (para teclados sin PrintScr) |
| `Print Screen` | Captura de pantalla |
| `Alt + Print Screen` | Grabación de pantalla |
| `Super + Print Screen` | Selector de color |
| `Alt + Shift + L` | Copiar URL actual de webapp o Chromium |
| `Super + Ctrl + X` | Iniciar/detener dictación (requiere *Instalar > AI > Dictación*) |

Con las grabaciones de pantalla, presiona el atajo para iniciar, presiona de nuevo para detener.

Todas las opciones de captura también están accesibles bajo *Captura* en el menú de Omarchy (`Super + Alt + Space`).

## Estilo

| Atajo | Función |
|-------|---------|
| `Super + Ctrl + Shift + Space` | Elegir nuevo tema |
| `Super + Ctrl + Space` | Elegir fondo de tema |
| `Super + Backspace` | Alternar transparencia en ventana |
| `Super + Ctrl + Backspace` | Alternar aspecto cuadrado de ventana única |

Las imágenes de fondo adicionales están en `~/.config/omarchy/current/backgrounds`. También disponibles via *Instalar > Fondo* en el menú de Omarchy.

Todas las opciones de estilo también están accesibles bajo *Estilo* en el menú de Omarchy (`Super + Alt + Space`).

## Tmux

La tecla prefijo es `Ctrl + Space` (`Ctrl + B` también funciona). Puedes cambiar estos atajos en `~/.config/tmux/tmux.conf`.

### Paneles

| Atajo | Función |
|-------|---------|
| `Prefijo + v` | Dividir panel al lado (vertical) |
| `Prefijo + h` | Dividir panel debajo (horizontal) |
| `Prefijo + x` | Matar panel |
| `Prefijo + z` | Alternar zoom de panel (pantalla completa) |
| `Ctrl + Alt + Flechas` | Mover entre paneles |
| `Ctrl + Alt + Shift + Flechas` | Redimensionar paneles |

### Ventanas

| Atajo | Función |
|-------|---------|
| `Prefijo + c` | Nueva ventana |
| `Prefijo + k` | Matar ventana |
| `Prefijo + r` | Renombrar ventana |
| `Alt + 1-9` | Ir a ventana específica |
| `Alt + Flecha Izquierda/Derecha` | Mover entre ventanas |

### Sesiones

| Atajo | Función |
|-------|---------|
| `Prefijo + C` | Nueva sesión |
| `Prefijo + K` | Matar sesión |
| `Prefijo + R` | Renombrar sesión |
| `Prefijo + N` | Siguiente sesión |
| `Prefijo + P` | Sesión anterior |
| `Alt + Flecha Arriba/Abajo` | Mover entre sesiones |
| `Prefijo + s` | Listar sesiones |
| `Prefijo + d` | Desconectar de sesión |

### Modo copia (estilo vi)

| Atajo | Función |
|-------|---------|
| `Prefijo + [` | Entrar al modo copia |
| `v` | Iniciar selección (en modo copia) |
| `y` | Copiar selección (en modo copia) |

### General

| Atajo | Función |
|-------|---------|
| `Prefijo + q` | Recargar configuración |
| `Prefijo + :` | Prompt de comando |

## Terminal Ghostty

Ghostty se instala usando *Instalar > Terminal* via el menú de Omarchy.

| Atajo | Función |
|-------|---------|
| `Ctrl + Shift + E` | Nueva división debajo |
| `Ctrl + Shift + O` | Nueva división al lado |
| `Ctrl + Alt + Flechas` | Mover entre divisiones |
| `Super + Ctrl + Shift + Flechas` | Redimensionar división por 10 líneas |
| `Super + Ctrl + Shift + Alt + Flechas` | Redimensionar división por 100 líneas |
| `Ctrl + Shift + T` | Nueva pestaña |
| `Ctrl + Shift + Flechas` | Mover entre pestañas |
| `Alt + Números` | Ir a pestaña específica |
| `Shift + Re Pag/Av Pag` | Desplazar historial |
| `Ctrl + Clic izquierdo` | Abrir enlace en navegador |

## Neovim (con LazyVim)

### Navegación

| Atajo | Función |
|-------|---------|
| `Space` | Mostrar opciones de comando |
| `Space Space` | Abrir archivo via búsqueda difusa |
| `Space E` | Alternar barra lateral |

| Atajo | Función |
|-------|---------|
