# Aplicaciones GUI

## Obsidian

[Obsidian](https://obsidian.md/) es una aplicación gratuita y altamente extensible para tomar notas que usa archivos Markdown simples para almacenamiento.

Obsidian es gratuito para todos los propósitos, incluyendo uso personal, comercial y sin fines de lucro.

Obsidian también ofrece un [complemento comercial para sincronización](https://obsidian.md/sync) con apps móviles en iOS y Android. (<https://obsidian.md/pricing>).

Inicias Obsidian con `Super + Shift + O`. Para usar sincronización de temas, debes seleccionar el tema `Omarchy` en configuración.

## Pinta

[Pinta](https://pinta-project.com/) es una herramienta básica de edición de imágenes genial para recortar, redimensionar y otras manipulaciones básicas. Solo no esperes una alternativa a Photoshop. Pero aún tiene Varita Mágica y capas.

Inicias Pinta via el lanzador de aplicaciones (`Super + Space`).

## LocalSend

[LocalSend](https://localsend.org/) te permite enviar archivos a otros dispositivos en la misma red que ejecutan la app, como AirDrop de Apple. Pero es multiplataforma, así que puedes enviar archivos a y desde Windows, macOS, Android, iOS y por supuesto Linux.

## LibreOffice

[LibreOffice](https://www.libreoffice.org/) es un paquete de oficina completo con procesador de texto, hoja de cálculo, presentaciones, aplicación de dibujo y más. Es compatible con archivos de Microsoft Office, así que esta es una gran forma de poder abrir esos documentos de Word.

Inicias LibreOffice via el lanzador de aplicaciones (`Super + Space`).

## Signal

[Signal](https://signal.org/) es el pionero del mensajería E2E encriptada, y una gran opción de comunicación para cualquiera que prefiera no pasar por uno de los grandes conglomerados tecnológicos.

Inicias Signal con `Super + Shift + G`.

## mpv

[mpv](https://mpv.io/) es un reproductor de medios simple y rápido que reproduce casi cualquier cosa desde cualquier fuente. Genial para ver videos.

Inicias mpv via el lanzador de aplicaciones (`Super + Space`) o simplemente haz doble clic en un video en el administrador de archivos.

## OBS Studio

[OBS Studio](https://obsproject.com/) te permite grabar o hacer streaming de video desde múltiples fuentes. Puedes mezclar una captura de pantalla con una webcam con entrada de micrófono. Es lo que se usó para grabar los screencasts de Omarchy.

Inicias OBS Studio via el lanzador de aplicaciones (`Super + Space`).

## Kdenlive

[Kdenlive](https://kdenlive.org/) es un excelente editor de video. Perfecto para trabajar con video que sale de OBS Studio antes de compartirlo.

Inicias Kdenlive via el lanzador de aplicaciones (`Super + Space`).

# Aplicaciones y servicios comerciales

Omarchy está mayormente enfocado en proporcionar software libre y de código abierto, pero no es religioso al respecto. A veces la mejor solución es una oferta comercial, y eso está bien. Aquí están algunas de las opciones para las que proporcionamos una instalación fácil.

## 1Password

Mantener tus contraseñas en un gestor de contraseñas es una mejor práctica. Más aún si estás trabajando con un equipo. Y [1password](https://1password.com/) es una gran solución, que también viene con una herramienta de línea de comandos para integrar búsquedas de claves en scripts.

Inicias 1Password con `Super + Shift + /`.

## Typora

[Typora](https://typora.io/) es una herramienta de escritura minimalista y sin distracciones en el mismo espíritu que iA Writer para Mac y Windows. Como Obsidian, usa Markdown para formateo, pero de una forma enfocada en escribir piezas individuales o ensayos y muy poco más.

Mejor usado en modo pantalla completa (F11) para ese look totalmente inmersivo de nada-más-palabras.

Viene con una prueba gratuita de 15 días, y luego es un costo único de $15.

Inicias Typora usando el lanzador de aplicaciones (`Super + Space`).

## Spotify

[Spotify](https://spotify.com/) es el servicio de streaming de música más popular del mundo. Y la aplicación de Linux proporciona todo lo que esperarías, incluyendo reproducción offline.

Inicias Spotify usando `Super + Shift + M`.

## Dropbox

[Dropbox](https://www.dropbox.com/) es una gran forma de sincronizar archivos entre máquinas mientras mantienes un backup en la nube. Para configurarlo, selecciona *Instalar > Servicio > Dropbox* desde el menú de Omarchy.

## Tailscale

[Tailscale](https://tailscale.com/) es un mesh VPN que hace que obtener acceso a todas tus computadoras y servidores por internet de forma segura sea super simple. Para configurarlo, selecciona *Instalar > Servicio > Tailscale* desde el menú de Omarchy.

## NordVPN

[NordVPN](https://nordvpn.com/) es un servicio VPN estándar que te permite salir con tu tráfico desde la mayoría de regiones del mundo. Para configurarlo, selecciona *Instalar > Servicio > NordVPN* desde el menú de Omarchy.
